# Lifeguard LLM Wiki

Living, LLM-maintained knowledge base for the Lifeguard ORM and migration tooling, per [Karpathy's llm-wiki pattern](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).

**If you are an agent starting a session, read in this order:**

1. [`AGENT.md`](../../AGENT.md) — strict rules at the repo root.
2. [`SCHEMA.md`](./SCHEMA.md) — how this wiki is structured and maintained.
3. [`index.md`](./index.md) — catalog of every page.
4. [`log.md`](./log.md) — chronological log; tail for recent context.
5. [`docs-catalog.md`](./docs-catalog.md) — inventory of authoritative docs the wiki synthesizes.
